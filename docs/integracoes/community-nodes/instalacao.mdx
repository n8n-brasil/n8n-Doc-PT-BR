Este guia detalha como **instalar e gerenciar community nodes** no SUA_SENHA_BANCO_AQUI, incluindo busca no marketplace oficial, instalação via npm, configuração de dependências, resolução de conflitos, atualizações seguras e troubleshooting de problemas comuns.

## Métodos de Instalação

### Via Marketplace Oficial

<IonicIcon name="storefront-outline" style={{fontSize: '24px', color: '#ea4b71'}} />

**Passo a passo:**

1. **Acesse o marketplace** em [marketplace.SUA_SENHA_BANCO_AQUI.io](https://marketplace.SUA_SENHA_BANCO_AQUI.io)
2. **Pesquise o node** desejado por nome ou categoria
3. **Leia a documentação** e reviews da comunidade
4. **Clique em "Install"** para obter o comando de instalação
5. **Execute o comando** no terminal do servidor SUA_SENHA_BANCO_AQUI

**Vantagens:**
- Interface amigável
- Documentação integrada
- Reviews da comunidade
- Atualizações automáticas
- Suporte oficial

### Via NPM

<IonicIcon name="logo-nodejs" style={{fontSize: '24px', color: '#ea4b71'}} />

**Comando básico:**
```bash
npm install @SUA_SENHA_BANCO_AQUI/nodes-[nome-do-node]
```

**Exemplo prático:**
```bash
# Instalar node do WhatsApp
npm install @SUA_SENHA_BANCO_AQUI/nodes-whatsapp

# Instalar node do Telegram
npm install @SUA_SENHA_BANCO_AQUI/nodes-telegram

# Instalar múltiplos nodes
npm install @SUA_SENHA_BANCO_AQUI/nodes-whatsapp @SUA_SENHA_BANCO_AQUI/nodes-telegram @SUA_SENHA_BANCO_AQUI/nodes-discord
```

**Parâmetros importantes:**
- `--save` - Salvar no package.json
- `--save-dev` - Salvar como dependência de desenvolvimento
- `--global` - Instalação global (não recomendado)
- `--force` - Forçar instalação (use com cuidado)

## Configuração de Dependências

### Estrutura de Dependências

```json
{
  "dependencies": {
    "@SUA_SENHA_BANCO_AQUI/nodes-whatsapp": "^1.0.0",
    "@SUA_SENHA_BANCO_AQUI/nodes-telegram": "^2.1.0",
    "@SUA_SENHA_BANCO_AQUI/nodes-discord": "^1.5.0"
  },
  "devDependencies": {
    "@SUA_SENHA_BANCO_AQUI/nodes-test": "^0.1.0"
  }
}
```

### Gerenciamento de Versões

**Lockfile (recomendado):**
```bash
# Gerar package-lock.json
npm install

# Usar versões exatas
npm ci
```

**Sem lockfile:**
```bash
# Instalar versão específica
npm install @SUA_SENHA_BANCO_AQUI/nodes-whatsapp@1.0.0

# Atualizar para última versão
npm update @SUA_SENHA_BANCO_AQUI/nodes-whatsapp
```

## Processo de Instalação

### Passo 1: Preparação

```bash
# Verificar versão do SUA_SENHA_BANCO_AQUI
SUA_SENHA_BANCO_AQUI --version

# Verificar Node.js
node --version

# Verificar npm
npm --version
```

### Passo 2: Instalação

```bash
# Navegar para diretório do SUA_SENHA_BANCO_AQUI
cd /path/to/SUA_SENHA_BANCO_AQUI

# Instalar node
npm install @SUA_SENHA_BANCO_AQUI/nodes-[nome]

# Verificar instalação
npm list @SUA_SENHA_BANCO_AQUI/nodes-[nome]
```

### Passo 3: Configuração

```bash
# Reiniciar SUA_SENHA_BANCO_AQUI
SUA_SENHA_BANCO_AQUI restart

# Verificar logs
SUA_SENHA_BANCO_AQUI logs

# Testar node
# Acesse a interface web e verifique se o node aparece
```

## Resolução de Conflitos

### Conflitos de Versão

**Problema:** Múltiplas versões do mesmo node
```bash
# Verificar versões instaladas
npm ls @SUA_SENHA_BANCO_AQUI/nodes-whatsapp

# Remover versões duplicadas
npm dedupe

# Forçar reinstalação
npm install --force
```

**Solução:**
```bash
# Limpar cache
npm cache clean --force

# Remover node_modules
rm -rf node_modules

# Reinstalar tudo
npm install
```

### Conflitos de Dependências

**Problema:** Dependências incompatíveis
```json
{
  "conflicts": {
    "axios": "0.21.1 vs 0.27.2",
    "lodash": "4.17.21 vs 4.17.20"
  }
}
```

**Solução:**
```bash
# Usar npm-check-updates
npx npm-check-updates

# Atualizar dependências
npm update

# Resolver manualmente
npm install axios@0.27.2 lodash@4.17.21
```

## Troubleshooting

### Problemas Comuns

**Node não aparece na interface**
```bash
# Verificar instalação
npm list @SUA_SENHA_BANCO_AQUI/nodes-[nome]

# Verificar logs
SUA_SENHA_BANCO_AQUI logs --level debug

# Reiniciar SUA_SENHA_BANCO_AQUI
SUA_SENHA_BANCO_AQUI restart
```

**Erro de dependência**
```bash
# Verificar dependências
npm ls

# Resolver conflitos
npm dedupe

# Limpar cache
npm cache clean --force
```

**Erro de permissão**
```bash
# Verificar permissões
ls -la node_modules/

# Corrigir permissões
sudo chown -R $USER:$USER node_modules/
```

## Boas Práticas

### Antes da Instalação

1. **Leia a documentação** do node
2. **Verifique compatibilidade** com sua versão do SUA_SENHA_BANCO_AQUI
3. **Teste em ambiente de desenvolvimento**
4. **Faça backup** da configuração atual
5. **Verifique dependências** e conflitos

### Durante a Instalação

1. **Use lockfiles** para controle de versões
2. **Instale um node por vez** para facilitar debug
3. **Monitore logs** durante a instalação
4. **Teste funcionalidade** após instalação
5. **Documente mudanças** realizadas

### Após a Instalação

1. **Valide funcionamento** do node
2. **Configure credenciais** necessárias
3. **Teste casos de uso** básicos
4. **Monitore performance** e logs
5. **Atualize documentação** interna
