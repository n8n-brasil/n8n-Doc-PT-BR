#ionicicon-namedocument-outline-style-fontsize-24px color: '#cor-ea4b71-google-sheets

A integração do Google Sheets é uma das mais populares do n8n, permitindo automatizar completamente suas planilhas e dados.

#ionicicon-namechevron-forward-outline-style-fontsize-24px color: '#cor-ea4b71-o-que você pode fazer

- **Ler dados** de planilhas existentes
- **Criar novas linhas** automaticamente
- **Atualizar células** específicas
- **Deletar dados** quando necessário
- **Criar novas planilhas** programaticamente
- **Gerenciar permissões** de acesso

#ionicicon-namegrid-outline-style-fontsize-24px color: '#cor-ea4b71-pr-requisitos

#id-1-conta-google-conta Google ativa
- Acesso ao Google Sheets

#id-2-credenciais-oauth2-criar projeto no Google Cloud Console
- Ativar Google Sheets API
- Configurar OAuth2 credentials

#ionicicon-namesettings-outline-style-fontsize-24px color: '#cor-ea4b71-configurao-rpida

#id-1-criar-credencial-1-v em:** Credentials → Add new
2. **Escolha:** Google OAuth2 API
3. **Configure:** Client ID e Client Secret
4. **Autorize:** Sua conta Google

#id-2-adicionar-node-1-busque "Google Sheets" no menu de nodes
2. **Arrastar:** Para o canvas
3. **Conectar:** Sua credencial OAuth2
