## **Como Implementar**

### Instalação
```bash
npm install react-bits
```

### Uso Básico
```jsx
import React from 'react';
import { AnimatedButton } from 'react-bits';

const MyComponent = () => {
return (
<AnimatedButton
variant="primary"
animation="bounce"
onClick={() => console.log('Clicado!')}
>
Clique Aqui
</AnimatedButton>
);
};
```

### Animações SVG
```jsx
import React from 'react';

const AnimatedSVG = () => {
return (
<svg width="200" height="200" viewBox="0 0 200 200">
<circle
cx="100"
cy="100"
r="50"
fill="var(--ifm-color-primary)"
className="animated-circle"
/>
<style jsx>{`
.animated-circle {
animation: pulse 2s infinite;
}
@keyframes pulse {
0% { transform: scale(1); }
50% { transform: scale(1.1); }
100% { transform: scale(1); }
}
`}</style>
</svg>
);
};
```

```css
}
```

## **Performance e Otimização**

### Melhores Práticas
1. **Lazy Loading**: Carregue componentes sob demanda
2. **Memoização**: Use React.memo() para componentes pesados
3. **Animações CSS**: Prefira CSS animations sobre JavaScript
4. **Viewport Detection**: Anime apenas elementos visíveis

### Exemplo de Otimização
```jsx
import React, { memo, lazy, Suspense } from 'react';

// Lazy loading do componente
const HeavyAnimatedComponent = lazy(() => 
import('./HeavyAnimatedComponent')
);

// Memoização para evitar re-renders desnecessários
const OptimizedComponent = memo(({ data }) => {
return (
<Suspense fallback={<div>Carregando...</div>}>
<HeavyAnimatedComponent data={data} />
</Suspense>
);
});
```
