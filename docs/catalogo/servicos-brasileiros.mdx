### Configuração de Credenciais

#### PIX JSON
```json
{
  "clientId": "...",
  "clientSecret": "...",
  "certificate": "path/to/cert.p12"
}
```

#### NFe.io JSON
```json
{
  "apiKey": "...",
  "environment": "production"
}
```

#### Receita Federal JSON
```json
{
  "cnpj": "...",
  "senha": "..."
}
```

```bash

NFE_API_KEY=...

# Receita Federal
RECEITA_CNPJ=...
RECEITA_SENHA=...
```
