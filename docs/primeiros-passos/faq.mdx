## Perguntas Básicas

### O que é o n8n?

O n8n é uma plataforma de automação de workflows que permite conectar diferentes aplicações e serviços através de uma interface visual intuitiva.

### O n8n é gratuito?

Sim, o n8n é open-source e gratuito para uso local. Existe também uma versão cloud paga com recursos adicionais.

### Preciso de conhecimento técnico?

Não necessariamente. O n8n oferece uma interface visual que facilita a criação de workflows, mas conhecimentos básicos de programação podem ser úteis para casos avançados.

## Instalação e Configuração

### Qual método de instalação escolher?

- **Iniciantes**: n8n Cloud (mais fácil)
- **Desenvolvedores**: Instalação local via npm
- **Empresas**: Self-hosted (mais controle)

### Posso migrar entre métodos de instalação?

Sim, você pode exportar seus workflows e migrar entre diferentes métodos de instalação.

### Quais são os requisitos mínimos?

- **Local**: Node.js 16+ e 2GB RAM
- **Self-hosted**: Servidor com Docker ou Node.js
- **Cloud**: Apenas um navegador

## Workflows e Automação

### Quantos workflows posso criar?

Não há limite no número de workflows que você pode criar.

### Posso agendar execuções?

Sim, o n8n suporta agendamento de workflows usando cron expressions.

### Como faço backup dos meus workflows?

Você pode exportar workflows individualmente ou fazer backup completo da instância.

## Segurança e Privacidade

### Meus dados ficam seguros?

Sim, o n8n oferece várias opções de segurança, incluindo criptografia e controle de acesso.

### Posso usar em ambiente corporativo?

Sim, o n8n é adequado para uso corporativo com recursos de segurança avançados.

### Como gerencio credenciais?

O n8n oferece gerenciamento seguro de credenciais com criptografia.
