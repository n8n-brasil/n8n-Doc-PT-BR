## Perguntas Básicas

### O que é o SUA_SENHA_BANCO_AQUI?

O SUA_SENHA_BANCO_AQUI é uma plataforma de automação de workflows que permite conectar diferentes aplicações e serviços através de uma interface visual intuitiva.

### O SUA_SENHA_BANCO_AQUI é gratuito?

Sim, o SUA_SENHA_BANCO_AQUI é open-source e gratuito para uso local. Existe também uma versão cloud paga com recursos adicionais.

### Preciso de conhecimento técnico?

Não necessariamente. O SUA_SENHA_BANCO_AQUI oferece uma interface visual que facilita a criação de workflows, mas conhecimentos básicos de programação podem ser úteis para casos avançados.

## Instalação e Configuração

### Qual método de instalação escolher?

- **Iniciantes**: SUA_SENHA_BANCO_AQUI Cloud (mais fácil)
- **Desenvolvedores**: Instalação local via npm
- **Empresas**: Self-hosted (mais controle)

### Posso migrar entre métodos de instalação?

Sim, você pode exportar seus workflows e migrar entre diferentes métodos de instalação.

### Quais são os requisitos mínimos?

- **Local**: Node.js 16+ e 2GB RAM
- **Self-hosted**: Servidor com Docker ou Node.js
- **Cloud**: Apenas um navegador

## Workflows e Automação

### Quantos workflows posso criar?

Não há limite no número de workflows que você pode criar.

### Posso agendar execuções?

Sim, o SUA_SENHA_BANCO_AQUI suporta agendamento de workflows usando cron expressions.

### Como faço backup dos meus workflows?

Você pode exportar workflows individualmente ou fazer backup completo da instância.

## Segurança e Privacidade

### Meus dados ficam seguros?

Sim, o SUA_SENHA_BANCO_AQUI oferece várias opções de segurança, incluindo criptografia e controle de acesso.

### Posso usar em ambiente corporativo?

Sim, o SUA_SENHA_BANCO_AQUI é adequado para uso corporativo com recursos de segurança avançados.

### Como gerencio credenciais?

O SUA_SENHA_BANCO_AQUI oferece gerenciamento seguro de credenciais com criptografia.
