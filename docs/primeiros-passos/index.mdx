### Já trabalha com automação ou integração?

1. **Vá direto para [Conectar Aplicações](/primeiros-passos/conectar-aplicacoes)** e veja as diferenças do n8n

## Conceitos Fundamentais

<IonicIcon name="library-outline" style={{fontSize: '24px', color: '#ea4b71'}} />

Para compreender completamente o potencial do n8n e como ele pode transformar seus processos, **recomendamos começar pelos [Conceitos Fundamentais](/primeiros-passos/conceitos-fundamentais)**.

<IonicIcon name="information-circle-outline" style={{fontSize: '18px', color: '#ea4b71'}} />
**Resumo rápido:**

O n8n é uma plataforma que permite conectar diferentes aplicações através de uma interface visual intuitiva. Você pode automatizar tarefas repetitivas, integrar sistemas e processar dados sem precisar escrever código para operações básicas.

<IonicIcon name="key-outline" style={{fontSize: '16px', color: '#fbbf24'}} />
**Conceitos-chave que você vai dominar:**

- **Workflows** - Sequências de automação visual
- **Nodes** - Blocos de funcionalidade específica
- **Triggers** - Pontos de partida para automações
- **Connections** - Como dados fluem entre sistemas
