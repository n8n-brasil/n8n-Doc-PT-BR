# <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Instalação Local via npm

A instalação local via **npm** é ideal para desenvolvimento, testes rápidos e experimentação offline. É a forma mais simples de começar com o n8n.

## <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Quando Usar npm?

### **Cenários Ideais:**

- **Desenvolvimento** - Criar e testar nodes customizados
- **Experimentos** - Testar integrações rapidamente
- **Aprendizado** - Estudar o n8n sem configurações complexas
- **Demonstrações** - Apresentar funcionalidades offline
- **Debug** - Investigar problemas localmente

### **Limitações:**

- **Sem HTTPS** - Webhooks externos podem não funcionar
- **Segurança básica** - Não recomendado para produção
- **Performance limitada** - Sem otimizações de produção
- **Dados locais** - Sem backup automático

---

## <IonicIcon name="grid-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Pré-requisitos

### **Node.js e npm**

```bash
# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Verificar versões
node --version  # Deve ser ≥ 20.19 ≤ 24.x
npm --version   # Qualquer versão recente

# <IonicIcon name="code-slash-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Instalar Node.js (se necessário)
# <IonicIcon name="sparkles-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> macOS
brew install node

# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Windows
# <IonicIcon name="code-slash-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Baixar de https://nodejs.org/
```

### **Requisitos do Sistema**

- **RAM**: Mínimo 2GB, recomendado 4GB+
- **Disco**: 1GB livre para instalação
- **Rede**: Conexão para download de dependências
- **Porta**: 5678 disponível (ou configurável)

## <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Instalação Rápida

### **1. Instalação Global (Recomendado)**

```bash
# <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Instalar n8n globalmente
npm install n8n -g

# <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Verificar instalação
n8n --version
```

### **2. Primeira Execução**

```bash
# <IonicIcon name="sparkles-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Iniciar n8n
n8n

# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Ou com comando explícito
n8n start
```

### **3. Acessar Interface**

Abra seu navegador e acesse: `http://localhost:5678`

## <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Configuração Avançada

### **Variáveis de Ambiente**

```bash
# <IonicIcon name="key-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Configurações básicas
export N8N_PORT=5678
export N8N_PROTOCOL=http
export N8N_HOST=localhost
export GENERIC_TIMEZONE=America/Sao_Paulo

# <IonicIcon name="key-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Executar com configurações
N8N_PORT=5678 \
N8N_PROTOCOL=http \
N8N_HOST=localhost \
GENERIC_TIMEZONE=America/Sao_Paulo \
n8n start
```

### **Arquivo .env**

Crie um arquivo `.env` no diretório onde executa o n8n:

```bash
# <IonicIcon name="key-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> .env
N8N_PORT=5678
N8N_PROTOCOL=http
N8N_HOST=localhost
GENERIC_TIMEZONE=America/Sao_Paulo
N8N_LOG_LEVEL=info
N8N_USER_MANAGEMENT_DISABLED=false
```

### **Configuração de Túnel (Webhooks)**

Para receber webhooks externos:

```bash
# <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Instalar ngrok
npm install -g ngrok

# <IonicIcon name="sparkles-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Em um terminal, iniciar túnel
ngrok http 5678

# <IonicIcon name="sparkles-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Em outro terminal, iniciar n8n com túnel
n8n start --tunnel
```

### **Instalação para Desenvolvimento**

```bash

git clone https://github.com/n8n-io/n8n.git
cd n8n

npm install

# <IonicIcon name="code-slash-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Executar em modo desenvolvimento
npm run dev
```

### **Criar Nodes Customizados**

```bash
# <IonicIcon name="code-slash-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Criar estrutura de node
mkdir my-custom-node
cd my-custom-node

# <IonicIcon name="sparkles-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Inicializar projeto
npm init -y

# <IonicIcon name="settings-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Instalar dependências do n8n
npm install n8n-core n8n-workflow

# <IonicIcon name="folder-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Estrutura básica
mkdir src
touch src/MyNode.ts
```

### **Exemplo de Node Customizado**

```typescript
// src/MyNode.ts
import { IExecuteFunctions } from 'n8n-core';
import { INodeExecutionData, INodeType, INodeTypeDescription } from 'n8n-workflow';

export class MyNode implements INodeType {
  description: INodeTypeDescription = {
    displayName: 'Meu Node',
    name: 'myNode',
    icon: 'file:myNode.svg',
    group: ['transform'],
    version: 1,
    description: 'Descrição do meu node customizado',
    defaults: {
      name: 'Meu Node',
    },
    inputs: ['main'],
    outputs: ['main'],
    properties: [
      {
        displayName: 'Operação',
        name: 'operation',
        type: 'string',
        default: 'process',
        description: 'Operação a ser executada',
      },
    ],
  };

  async execute(this: IExecuteFunctions): Promise<INodeExecutionData[][]> {
    const items = this.getInputData();
    const returnData: INodeExecutionData[] = [];

    for (let i = 0; i < items.length; i++) {
      const operation = this.getNodeParameter('operation', i) as string;
      
      // Sua lógica aqui
      const processedData = {
        ...items[i].json,
        processed: true,
        operation,
      };

      returnData.push({
        json: processedData,
      });
    }

    return [returnData];
  }
}
```

## <IonicIcon name="bug-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Debug e Troubleshooting

### **Logs Detalhados**

```bash
# <IonicIcon name="bug-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Logs de debug
N8N_LOG_LEVEL=debug n8n start

# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Logs específicos
N8N_LOG_LEVEL=error n8n start

# <IonicIcon name="grid-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Salvar logs em arquivo
N8N_LOG_LEVEL=debug n8n start > n8n.log 2>&1
```

### **Problemas Comuns**

#### **Porta já em uso**

```bash
# <IonicIcon name="git-branch-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Verificar processo usando a porta
netstat -tulpn | grep 5678
lsof -i :5678

# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Usar porta diferente
N8N_PORT=5679 n8n start
```

#### **Permissões (Linux/Mac)**

```bash
# <IonicIcon name="shield-checkmark-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Corrigir permissões
sudo chown -R $(whoami) ~/.n8n
chmod -R 755 ~/.n8n
```

#### **Erro de memória**

```bash
# <IonicIcon name="sparkles-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Aumentar limite de memória
NODE_OPTIONS="--max-old-space-size=4096" n8n start
```

#### **Problemas de rede**

```bash
# <IonicIcon name="document-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Verificar conectividade
curl -I http://localhost:5678

# <IonicIcon name="git-network-outline" style={{ fontSize: '24px', color: '#ea4b71' }} /> Testar webhooks
curl -X POST http://localhost:5678/webhook/test
```
