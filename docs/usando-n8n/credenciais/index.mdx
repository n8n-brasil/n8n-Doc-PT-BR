#ionicicon-namesparkles-outline-style-fontsize-24px color: '#cor-ea4b71-conceitos-fundamentais

#o-que-so-credenciais-credenciais são informações de autenticação que permitem ao n8n se conectar com aplicações externas de forma segura. Elas podem incluir:

- Chaves de API
- Nomes de usuário e senhas
- Tokens de acesso
- Certificados SSL

#segurana-das-credenciais-o-n8n oferece várias camadas de segurança:

- **Criptografia**: Todas as credenciais são criptografadas
- **Isolamento**: Cada usuário vê apenas suas próprias credenciais
- **Compartilhamento Controlado**: Compartilhe credenciais apenas com quem precisa
