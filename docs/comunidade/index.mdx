# <IonicIcon name="people-outline" size={32} color="#ea4b71" /> Comunidade n8n Brasil

A comunidade n8n Brasil é um ponto de encontro para quem automatiza com contexto local. Aqui você encontra tutoriais em português, integrações com serviços nacionais, vídeos, projetos de código aberto e formas acessíveis de começar ou expandir suas automações.

:::tip **Importante**
Esta iniciativa é independente e comunitária. Nossa missão é complementar a documentação oficial do n8n, tornando o conteúdo mais acessível e relevante para pessoas desenvolvedoras no Brasil.
:::
