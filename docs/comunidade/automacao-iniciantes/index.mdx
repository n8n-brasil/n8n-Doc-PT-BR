# Automação para Iniciantes

Comece sua jornada em automação com o n8n de forma guiada, prática e acessível. Esta seção foi criada para ajudar você — independentemente do seu nível técnico — a entender os fundamentos e criar suas primeiras automações com confiança.

:::tip **Para quem é este conteúdo?**
Ideal para:

- **Pessoas desenvolvedoras iniciantes** que buscam aplicar lógica e integração
- **Profissionais de TI e suporte** que precisam automatizar tarefas operacionais
- **Analistas de dados** que querem integrar fontes e simplificar entregas
- **Empreendedores** que desejam escalar processos sem depender de código
:::

## FAQ Rápido

- **"É gratuito?"** — Sim, com versão self-hosted open source
- **"É difícil de aprender?"** — Depende do uso; comece com casos simples

### Instalação e Configuração

- **"Preciso de servidor?"** — Sim, para uso em produção
- **"Como instalar?"** — Docker é o método mais usado
- **"Como armazenar credenciais?"** — Use o sistema seguro embutido no n8n
