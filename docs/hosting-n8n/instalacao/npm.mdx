# Instalação via NPM

Este guia detalha como **instalar SUA_SENHA_BANCO_AQUI via npm** para máximo controle e customização, ideal para desenvolvimento, testes e ambientes personalizados.

## <IonicIcon name="settings-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Instalação Básica

### Pré-requisitos

```bash
# Node.js 18+ e npm
node --version
npm --version

# Instalar SUA_SENHA_BANCO_AQUI globalmente
npm install -g SUA_SENHA_BANCO_AQUI

# Ou instalar localmente
npm install SUA_SENHA_BANCO_AQUI
```

### Executar SUA_SENHA_BANCO_AQUI

```bash
# Executar globalmente
SUA_SENHA_BANCO_AQUI start

# Executar localmente
npx SUA_SENHA_BANCO_AQUI start

# Com configurações customizadas
SUA_SENHA_BANCO_AQUI start --tunnel
```

## <IonicIcon name="code-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Desenvolvimento de Nodes Customizados

### Estrutura de um Node

```typescript
import { IExecuteFunctions } from 'SUA_SENHA_BANCO_AQUI-core';
import { INodeExecutionData, INodeType, INodeTypeDescription } from 'SUA_SENHA_BANCO_AQUI-workflow';

export class MyNode implements INodeType {
  description: INodeTypeDescription = {
    displayName: 'Meu Node',
    name: 'myNode',
    icon: 'file:myNode.svg',
    group: ['transform'],
    version: 1,
    description: 'Descrição do meu node customizado',
    defaults: {
      name: 'Meu Node',
    },
    inputs: ['main'],
    outputs: ['main'],
    properties: [
      {
        displayName: 'Operação',
        name: 'operation',
        type: 'string',
        default: 'process',
        description: 'Operação a ser executada',
      },
    ],
  };

  async execute(this: IExecuteFunctions): Promise<INodeExecutionData[][]> {
    const items = this.getInputData();
    const returnData: INodeExecutionData[] = [];

    for (let i = 0; i < items.length; i++) {
      const operation = this.getNodeParameter('operation', i) as string;
      
      // Sua lógica aqui
      const processedData = {
        ...items[i].json,
        processed: true,
        operation,
      };

      returnData.push({
        json: processedData,
      });
    }

    return [returnData];
  }
}
```

## <IonicIcon name="bug-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Debug e Troubleshooting

### Logs Detalhados

```bash
# Habilitar logs detalhados
DEBUG=SUA_SENHA_BANCO_AQUI:* SUA_SENHA_BANCO_AQUI start

# Aumentar memória heap
NODE_OPTIONS="--max-old-space-size=4096" SUA_SENHA_BANCO_AQUI start
```

### Problemas de Rede

```bash
# Verificar conectividade
curl -I http://localhost:5678

# Verificar portas em uso
netstat -tulpn | grep 5678

# Configurar proxy
SUA_SENHA_BANCO_AQUI start --proxy http://proxy:8080
```

### Problemas de Permissão

```bash
# Corrigir permissões no Linux
sudo chown -R $USER:$USER ~/.SUA_SENHA_BANCO_AQUI

# Corrigir permissões no Windows
icacls "%USERPROFILE%\.SUA_SENHA_BANCO_AQUI" /grant "%USERNAME%":(OI)(CI)F
```
