# Instalação via Docker

Este guia mostrará como instalar e executar o SUA_SENHA_BANCO_AQUI usando Docker, a forma mais robusta e recomendada para ambientes de produção.

## <IonicIcon name="school-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Configuração Básica

### Docker Compose Simples

```yaml
version: '3.8'

services:
  SUA_SENHA_BANCO_AQUI:
    image: SUA_SENHA_BANCO_AQUIio/SUA_SENHA_BANCO_AQUI:latest
    restart: unless-stopped
    ports:
      - "5678:5678"
    environment:
      - N8N_PROTOCOL=https
      - N8N_HOST=seu-dominio.com
      - GENERIC_TIMEZONE=America/Sao_Paulo
      - WEBHOOK_URL=https://seu-dominio.com/
    volumes:
      - SUA_SENHA_BANCO_AQUI_data:/home/node/.SUA_SENHA_BANCO_AQUI

volumes:
  SUA_SENHA_BANCO_AQUI_data:
```

## <IonicIcon name="server-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Configuração Completa com Banco de Dados

```yaml
version: '3.8'

services:
  SUA_SENHA_BANCO_AQUI:
    image: SUA_SENHA_BANCO_AQUIio/SUA_SENHA_BANCO_AQUI:latest
    restart: unless-stopped
    ports:
      - "5678:5678"
    environment:
      - N8N_PROTOCOL=https
      - N8N_HOST=seu-dominio.com
      - GENERIC_TIMEZONE=America/Sao_Paulo
      - DB_TYPE=postgresdb
      - DB_POSTGRESDB_HOST=postgres
      - DB_POSTGRESDB_PORT=5432
      - DB_POSTGRESDB_DATABASE=SUA_SENHA_BANCO_AQUI
      - DB_POSTGRESDB_USER=SUA_SENHA_BANCO_AQUI
      - DB_POSTGRESDB_PASSWORD=SUA_SENHA_SEGURA_AQUI
      - REDIS_URL=redis://redis:6379
      - N8N_ENCRYPTION_KEY=sua_chave_32_caracteres
    volumes:
      - SUA_SENHA_BANCO_AQUI_data:/home/node/.SUA_SENHA_BANCO_AQUI
    depends_on:
      - postgres
      - redis

  postgres:
    image: postgres:15
    restart: unless-stopped
    environment:
      - POSTGRES_DB=SUA_SENHA_BANCO_AQUI
      - POSTGRES_USER=SUA_SENHA_BANCO_AQUI
      - POSTGRES_PASSWORD=SUA_SENHA_SEGURA_AQUI
    volumes:
      - postgres_data:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    restart: unless-stopped
    command: redis-server --appendonly yes
    volumes:
      - redis_data:/data

volumes:
  SUA_SENHA_BANCO_AQUI_data:
  postgres_data:
  redis_data:
```

## <IonicIcon name="sparkles-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Executar com Docker Compose

```bash
# Baixar e executar
docker-compose up -d

# Ver logs
docker-compose logs -f SUA_SENHA_BANCO_AQUI

# Parar
docker-compose down
```

## <IonicIcon name="document-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Configuração Nginx

### nginx.conf

```nginx
events {
    worker_connections 1024;
}

http {
    upstream SUA_SENHA_BANCO_AQUI_backend {
        server SUA_SENHA_BANCO_AQUI:5678;
    }

    server {
        listen 80;
        server_name seu-dominio.com;
        return 301 https://$server_name$request_uri;
    }

    server {
        listen 443 ssl http2;
        server_name seu-dominio.com;

        ssl_certificate /etc/nginx/ssl/cert.pem;
        ssl_certificate_key /etc/nginx/ssl/key.pem;

        location / {
            proxy_pass http://SUA_SENHA_BANCO_AQUI_backend;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;
        }
    }
}
```

### Configuração de Filas com Redis

```yaml
services:
  SUA_SENHA_BANCO_AQUI:
    environment:
      - REDIS_URL=redis://redis:6379
      - EXECUTIONS_PROCESS=main
      - EXECUTIONS_MODE=regular
      - EXECUTIONS_TIMEOUT=300000
      - EXECUTIONS_TIMEOUT_MAX=3600000

  redis:
    image: redis:7-alpine
    command: redis-server --appendonly yes --requirepass ${REDIS_PASSWORD}
    volumes:
      - redis_data:/data
```

### Load Balancing com Múltiplas Instâncias

```yaml
version: '3.8'

services:
  SUA_SENHA_BANCO_AQUI-1:
    image: SUA_SENHA_BANCO_AQUIio/SUA_SENHA_BANCO_AQUI:latest
    environment:
      - EXECUTIONS_PROCESS=main
      - REDIS_URL=redis://redis:6379
      - DB_TYPE=postgresdb
      - DB_POSTGRESDB_HOST=postgres
    depends_on:
      - postgres
      - redis

  SUA_SENHA_BANCO_AQUI-2:
    image: SUA_SENHA_BANCO_AQUIio/SUA_SENHA_BANCO_AQUI:latest
    environment:
      - EXECUTIONS_PROCESS=main
      - REDIS_URL=redis://redis:6379
      - DB_TYPE=postgresdb
      - DB_POSTGRESDB_HOST=postgres
    depends_on:
      - postgres
      - redis

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx-lb.conf:/etc/nginx/nginx.conf
    depends_on:
      - SUA_SENHA_BANCO_AQUI-1
      - SUA_SENHA_BANCO_AQUI-2
```

## <IonicIcon name="chevron-forward-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Backup e Monitoramento

### Script de Backup

```bash
#!/bin/bash
BACKUP_DIR="/backups/SUA_SENHA_BANCO_AQUI"
DATE=$(date +%Y%m%d_%H%M%S)

# Backup do banco de dados
docker exec postgres pg_dump -U SUA_SENHA_BANCO_AQUI SUA_SENHA_BANCO_AQUI > $BACKUP_DIR/db_$DATE.sql

# Backup dos dados do SUA_SENHA_BANCO_AQUI
docker exec SUA_SENHA_BANCO_AQUI tar -czf - /home/node/.SUA_SENHA_BANCO_AQUI > $BACKUP_DIR/SUA_SENHA_BANCO_AQUI_$DATE.tar.gz

# Limpar backups antigos (mais de 7 dias)
find $BACKUP_DIR -name "*.tar.gz" -mtime +7 -delete
find $BACKUP_DIR -name "*.sql" -mtime +7 -delete
```

### Cron Job para Backup

```bash
# Adicionar ao crontab
0 2 * * * /path/to/backup_script.sh
```

### Health Check

```bash
#!/bin/bash
CONTAINER_NAME="SUA_SENHA_BANCO_AQUI"
HEALTH_URL="http://localhost:5678/healthz"

if ! docker ps | grep -q $CONTAINER_NAME; then
    echo "Container $CONTAINER_NAME não está rodando"
    docker-compose up -d
    exit 1
fi

RESPONSE=$(curl -s -o /dev/null -w "%{http_code}" $HEALTH_URL)

if [ $RESPONSE -ne 200 ]; then
    echo "SUA_SENHA_BANCO_AQUI não está respondendo: $RESPONSE"
    docker-compose restart SUA_SENHA_BANCO_AQUI
fi
```

## <IonicIcon name="bug-outline" style={{fontSize: '24px', color: '#ea4b71'}} /> Troubleshooting

### Problemas Comuns

#### Container não inicia

```bash
# Verificar logs
docker-compose logs SUA_SENHA_BANCO_AQUI

# Verificar se a porta está em uso
netstat -tulpn | grep 5678

# Reiniciar container
docker-compose restart SUA_SENHA_BANCO_AQUI
```

#### Problemas de Permissão

```bash
# Corrigir permissões do volume
sudo chown -R 1000:1000 /path/to/SUA_SENHA_BANCO_AQUI_data
```

#### Problemas de Rede

```bash
# Verificar conectividade
docker exec SUA_SENHA_BANCO_AQUI ping postgres
docker exec SUA_SENHA_BANCO_AQUI ping redis

# Verificar variáveis de ambiente
docker exec SUA_SENHA_BANCO_AQUI env | grep N8N
```
