## <LocalIcon name="construct-outline" size={24} color="#ef4444" /> Problemas Técnicos

### **Antes de Pedir Ajuda**
1. **Pesquise** na documentação oficial
2. **Verifique** se já existe issue no GitHub
3. **Teste** em ambiente isolado
4. **Documente** o problema com detalhes

### **Como Reportar um Bug**
```markdown
**Descrição do Problema:**
[Descreva o que aconteceu]

**Passos para Reproduzir:**
1. Acesse...
2. Clique em...
3. Configure...

**Comportamento Esperado:**
[O que deveria acontecer]

**Comportamento Atual:**
[O que está acontecendo]

**Informações do Sistema:**
- n8n Version: [versão]
- Node.js Version: [versão]
- Sistema Operacional: [SO]
- Navegador: [se aplicável]

**Logs de Erro:**
[Cole os logs relevantes]
```

## <LocalIcon name="time-outline" size={24} color="#6b7280" /> Tempos de Resposta

### **Urgência Alta (Crítico)**
- **Canal**: Discord/Telegram
- **Tempo**: 2-4 horas
- **Uso**: Sistema em produção parado

### **Urgência Média (Importante)**
- **Canal**: GitHub Issues
- **Tempo**: 24-48 horas
- **Uso**: Funcionalidade não funciona

### **Urgência Baixa (Dúvida)**
- **Canal**: GitHub Discussions
- **Tempo**: 1-3 dias
- **Uso**: Perguntas gerais, sugestões

## <LocalIcon name="heart-outline" size={24} color="#ea4b71" /> Como Contribuir com o Suporte

### **Ajude Outros**
- Responda perguntas no Discord/Telegram

### **Melhore a Documentação**
- Reporte erros na documentação

- Compartilhe seus workflows
- Ajude a organizar eventos
- Divulgue o n8n no Brasil
