# <LocalIcon name="color-palette-outline" size={32} color="#ea4b71" /> Guia de Estilo

Este guia estabelece os **padrões de estilo e formatação** para manter consistência na documentação n8n Brasil.

---

## <LocalIcon name="chatbubbles-outline" size={24} color="#10b981" /> Tom de Voz

### **Princípios Gerais**
- **Técnico e preciso**: Use linguagem clara e técnica
- **Inclusivo**: Evite jargões desnecessários e seja acessível
- **Profissional**: Mantenha um tom institucional mas amigável
- **Contextualizado**: Adapte exemplos para o contexto brasileiro

### **Linguagem Inclusiva**
- Use "pessoas desenvolvedoras" em vez de "desenvolvedores"
- Prefira "usuários" ou "pessoas usuárias" 
- Evite gêneros específicos quando possível
- Use "você" para criar proximidade

---

## <LocalIcon name="text-outline" size={24} color="#ea4b71" /> Formatação

### **Títulos e Subtítulos**
```markdown
# Título Principal (H1)
## Seção Principal (H2) 
### Subseção (H3)
#### Tópico Específico (H4)
```

### **Destaques e Alertas**
```markdown
:::tip **Dica Importante**
Conteúdo útil para o usuário
:::

:::warning **Atenção**
Aviso sobre algo importante
:::

:::danger **Cuidado**
Informação crítica ou perigosa
:::

:::info **Informação**
Dados adicionais ou contexto
:::
```

### **Listas**
```markdown
- Item simples
- **Item em negrito**
- Item com [link](./exemplo)

1. Item numerado
2. **Item numerado em negrito**
3. Item com `código inline`
```

## <LocalIcon name="grid-outline" size={24} color="#8b5cf6" /> Tabelas

### **Estrutura Padrão**
```markdown
| Campo | Tipo | Descrição | Obrigatório |
|-------|------|-----------|-------------|
| `url` | string | URL do webhook | Sim |
| `method` | string | Método HTTP | Não |
| `headers` | object | Cabeçalhos customizados | Não |
```

### **Tabelas de Integração**
Para integrações, use sempre:
- **Campo**: Nome do campo/parâmetro
- **Tipo**: Tipo de dado esperado
- **Descrição**: Explicação clara do propósito
- **Obrigatório**: Sim/Não

## <LocalIcon name="image-outline" size={24} color="#06b6d4" /> Imagens e Screenshots

### **Padrões de Nomenclatura**
```
interface-n8n.png
workflow-exemplo.png
configuracao-webhook.png
```

### **Formatação**
```markdown
![Descrição da imagem](./caminho/para/imagem.png)

**Figura 1:** Interface principal do n8n
```

### **Requisitos**
- Use formato PNG ou JPG
- Mantenha resolução adequada (mínimo 800px de largura)
- Inclua descrição acessível
- Otimize o tamanho do arquivo
