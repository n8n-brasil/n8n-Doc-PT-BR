# <ion-icon name="document-text-outline" style={{ fontSize: '32px', color: '#ea4b71' }}></ion-icon> Guia de Estilo

Este guia estabelece os **padrões de estilo e formatação** para manter consistência na documentação n8n Brasil.

---

## <ion-icon name="chevron-forward-outline" style={{ fontSize: '24px', color: '#ea4b71' }}></ion-icon> Tom de Voz

### **Princípios Gerais**
- **Técnico e preciso**: Use linguagem clara e técnica
- **Inclusivo**: Evite jargões desnecessários e seja acessível
- **Profissional**: Mantenha um tom institucional mas amigável
- **Contextualizado**: Adapte exemplos para o contexto brasileiro

### **Linguagem Inclusiva**
- Use "pessoas desenvolvedoras" em vez de "desenvolvedores"
- Prefira "usuários" ou "pessoas usuárias" 
- Evite gêneros específicos quando possível
- Use "você" para criar proximidade

---

## <ion-icon name="repeat-outline" style={{ fontSize: '24px', color: '#ea4b71' }}></ion-icon> Formatação

### **Títulos e Subtítulos**
```markdown
# <ion-icon name="document-outline" style={{ fontSize: '32px', color: '#ea4b71' }}></ion-icon> Título Principal (H1)
## <ion-icon name="chevron-forward-outline" style={{ fontSize: '24px', color: '#ea4b71' }}></ion-icon> Seção Principal (H2)
### Subseção (H3)
#### Tópico Específico (H4)
```

### **Destaques e Alertas**
```markdown
:::tip **Dica Importante**
Conteúdo útil para o usuário
:::

:::warning **Atenção**
Aviso sobre algo importante
:::

:::danger **Cuidado**
Informação crítica ou perigosa
:::

:::info **Informação**
Dados adicionais ou contexto
:::
```

### **Listas**
```markdown
- Item simples
- **Item em negrito**
- Item com [link](./exemplo)

1. Item numerado
2. **Item numerado em negrito**
3. Item com `código inline`
```

## <ion-icon name="chevron-forward-outline" style={{ fontSize: '24px', color: '#ea4b71' }}></ion-icon> Tabelas

### **Estrutura Padrão**
```markdown
| Campo | Tipo | Descrição | Obrigatório |
|-------|------|-----------|-------------|
| `url` | string | URL do webhook | Sim |
| `method` | string | Método HTTP | Não |
| `headers` | object | Cabeçalhos customizados | Não |
```

### **Tabelas de Integração**
Para integrações, use sempre:
- **Campo**: Nome do campo/parâmetro
- **Tipo**: Tipo de dado esperado
- **Descrição**: Explicação clara do propósito
- **Obrigatório**: Sim/Não

## <ion-icon name="grid-outline" style={{ fontSize: '24px', color: '#ea4b71' }}></ion-icon> Imagens e Screenshots

### **Padrões de Nomenclatura**
```
interface-n8n.png
workflow-exemplo.png
configuracao-webhook.png
```

### **Formatação**
```markdown
![Descrição da imagem](./caminho/para/imagem.png)

**Figura 1:** Interface principal do n8n
```

### **Requisitos**
- Use formato PNG ou JPG
- Mantenha resolução adequada (mínimo 800px de largura)
- Inclua descrição acessível
- Otimize o tamanho do arquivo

## <ion-icon name="chevron-forward-outline" style={{ fontSize: '24px', color: '#ea4b71' }}></ion-icon> Checklist de Qualidade

Antes de submeter uma contribuição, verifique:

- [ ] **Tom de voz** adequado e inclusivo
- [ ] **Formatação** consistente com o guia
- [ ] **Links** funcionando e atualizados
- [ ] **Código** bem formatado e testado
- [ ] **Imagens** com descrição acessível
- [ ] **Tabelas** seguindo o padrão estabelecido
- [ ] **Componentes** usando a sintaxe correta
- [ ] **Contextualização** para o Brasil quando aplicável
