# <ion-icon name="information-circle-outline" style={{ fontSize: '32px', color: '#ea4b71' }}></ion-icon> Introdução

Bem-vindo à nossa central de colaboração! A força do n8n vem de sua comunidade vibrante, e existem duas maneiras fantásticas de fazer parte dela:

:::tip <ion-icon name="help-circle-outline" style={{ fontSize: '18px' }}></ion-icon> Qual é a diferença?
Pense assim:
* **Documentação Brasileira:** É como escrever um **guia turístico local** em português para uma cidade global. Você ajuda os brasileiros a navegar melhor.
* **Projeto Oficial n8n:** É como **ajudar a construir** os prédios e ruas dessa cidade global. Você melhora o produto para todos no mundo.
:::

---

### <ion-icon name="location-outline" style={{ fontSize: '20px' }}></ion-icon> Contribuir com a Documentação Brasileira (Nossa Iniciativa)

Ajude a criar o **melhor guia de n8n em português**. Esta é uma iniciativa da comunidade, para a comunidade.

- <ion-icon name="create-outline" style={{ fontSize: '16px' }}></ion-icon> **O que você faz?** Escreve tutoriais, traduz, corrige textos, cria exemplos com foco no Brasil.
- <ion-icon name="trending-up-outline" style={{ fontSize: '16px' }}></ion-icon> **Impacto:** Ajuda milhares de brasileiros a aprender e usar o n8n mais rápido.
- <ion-icon name="people-outline" style={{ fontSize: '16px' }}></ion-icon> **É oficial?** Não, somos um projeto da comunidade, complementar à documentação oficial.

<ion-icon name="arrow-forward-outline" style={{ fontSize: '16px' }}></ion-icon> [Quero ajudar a documentação brasileira!](./esta-documentacao/)

---

### <ion-icon name="globe-outline" style={{ fontSize: '20px' }}></ion-icon> Contribuir com o Projeto Oficial n8n (Global)

Participe do **desenvolvimento do n8n** que é usado por pessoas no mundo todo.

- <ion-icon name="code-outline" style={{ fontSize: '16px' }}></ion-icon> **O que você faz?** Contribui com código, cria novos nodes, reporta bugs, melhora a documentação oficial (em inglês).
- <ion-icon name="planet-outline" style={{ fontSize: '16px' }}></ion-icon> **Impacto:** Suas mudanças afetam o produto globalmente.
- <ion-icon name="checkmark-circle-outline" style={{ fontSize: '16px' }}></ion-icon> **É oficial?** Sim, estas são as diretrizes para contribuir diretamente com a empresa n8n.

<ion-icon name="arrow-forward-outline" style={{ fontSize: '16px' }}></ion-icon> [Quero contribuir para o projeto oficial!](./n8n-oficial/)

### <ion-icon name="help-circle-outline" style={{ fontSize: '20px' }}></ion-icon> Perguntas Orientadoras

- **Você tem experiência com n8n?** Se sim, considere [contribuir com tutoriais avançados](./esta-documentacao/). Se não, comece [reportando erros ou sugerindo melhorias](./esta-documentacao/01-entendendo-o-projeto/como-contribuir).
- **Prefere escrever ou programar?** Para escrita, vá para [documentação brasileira](./esta-documentacao/). Para código, explore [contribuições técnicas](./n8n-oficial/).
- **Quanto tempo tem disponível?** Para contribuições rápidas, veja [correções simples](./esta-documentacao/01-entendendo-o-projeto/como-contribuir). Para projetos maiores, confira [desenvolvimento de recursos](./n8n-oficial/contribuir-codigo-e-docs).
