# <LocalIcon name="heart-outline" size={32} /> Introdução

Bem-vindo à nossa central de colaboração! A força do n8n vem de sua comunidade vibrante, e existem duas maneiras fantásticas de fazer parte dela:

:::tip <LocalIcon name="help-circle-outline" size={18} /> Qual é a diferença?
Pense assim:
* **Documentação Brasileira:** É como escrever um **guia turístico local** em português para uma cidade global. Você ajuda os brasileiros a navegar melhor.
* **Projeto Oficial n8n:** É como **ajudar a construir** os prédios e ruas dessa cidade global. Você melhora o produto para todos no mundo.
:::

---

### <LocalIcon name="location-outline" size={20} /> Contribuir com a Documentação Brasileira (Nossa Iniciativa)

Ajude a criar o **melhor guia de n8n em português**. Esta é uma iniciativa da comunidade, para a comunidade.

- <LocalIcon name="create-outline" size={16} /> **O que você faz?** Escreve tutoriais, traduz, corrige textos, cria exemplos com foco no Brasil.
- <LocalIcon name="trending-up-outline" size={16} /> **Impacto:** Ajuda milhares de brasileiros a aprender e usar o n8n mais rápido.
- <LocalIcon name="people-outline" size={16} /> **É oficial?** Não, somos um projeto da comunidade, complementar à documentação oficial.

<LocalIcon name="arrow-forward-outline" size={16} /> [Quero ajudar a documentação brasileira!](./esta-documentacao/index)

---

### <LocalIcon name="globe-outline" size={20} /> Contribuir com o Projeto Oficial n8n (Global)

Participe do **desenvolvimento do n8n** que é usado por pessoas no mundo todo.

- <LocalIcon name="code-outline" size={16} /> **O que você faz?** Contribui com código, cria novos nodes, reporta bugs, melhora a documentação oficial (em inglês).
- <LocalIcon name="planet-outline" size={16} /> **Impacto:** Suas mudanças afetam o produto globalmente.
- <LocalIcon name="checkmark-circle-outline" size={16} /> **É oficial?** Sim, estas são as diretrizes para contribuir diretamente com a empresa n8n.

<LocalIcon name="arrow-forward-outline" size={16} /> [Quero contribuir para o projeto oficial!](./n8n-oficial/index)

## <LocalIcon name="compass-outline" size={28} /> Não Sabe Por Onde Começar?

Explore estas perguntas para encontrar o caminho de contribuição perfeito para você.

### <LocalIcon name="help-circle-outline" size={20} /> Perguntas Orientadoras

- **Você tem experiência com n8n?** Se sim, considere [contribuir com tutoriais avançados](./esta-documentacao/index). Se não, comece [reportando erros ou sugerindo melhorias](./esta-documentacao/01-entendendo-o-projeto/como-contribuir).
- **Prefere escrever ou programar?** Para escrita, vá para [documentação brasileira](./esta-documentacao/index). Para código, explore [contribuições técnicas](./n8n-oficial/index).
- **Quanto tempo tem disponível?** Para contribuições rápidas, veja [correções simples](./esta-documentacao/01-entendendo-o-projeto/como-contribuir). Para projetos maiores, confira [desenvolvimento de recursos](./n8n-oficial/contribuir-codigo-e-docs).
