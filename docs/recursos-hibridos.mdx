# Configuração e Uso

## Como Usar

1. **Importe o componente**:
   ```jsx
   import HybridProcessor from '@site/src/components/HybridProcessor';
   ```

2. **Use com tipo específico**:
   ```jsx
   <HybridProcessor 
     type="rst" 
     content={`
   .. note:: Exemplo
      Conteúdo RST aqui.
   `}    />
   ```

3. **Detecção automática**:
   ```jsx
   <HybridProcessor 
     content={`
   .. tip:: Detecção
      O tipo será detectado automaticamente.
   `}    />
   ```

## Tipos Suportados

- **`rst`**: reStructuredText puro
- **`sphinx`**: Extensões Sphinx
- **`auto`**: Detecção automática (padrão)
